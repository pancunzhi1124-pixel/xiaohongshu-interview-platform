## 结构化面试答案生成与排版规范

1. 所有 structured-answer-overrides*.json 答案必须使用清晰换行。
2. “答题思路、核心得分点、完整参考答案、评分标准”必须分段。
3. “第一，第二，第三，第四，首先，其次，再次，最后，一是，二是，三是”必须另起一行。
4. 答案要像多年公考、事业单位、国企央企、银行结构化面试教研老师写的示范答案。
5. 每题必须先判断题型，再结合题干写答案。
6. 禁止模板化、空泛化、堆砌口号。
7. 不要修改 structured-interview-questions.json。
8. 修改 JSON 后必须运行 JSON 校验。
9. 页面答案展示容器必须保留换行，例如 whitespace-pre-line。
10. 大量答案生成时，每 20 题一个 JSON 文件，每批自检后再继续。
